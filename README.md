# Glacial Gems OS

This repository is the internal operating system (“OS”) for **Glacial Gems ehf**.

It acts as the single source of truth for how the company thinks, operates, and documents its work.  
Both humans and AI assistants may read this repository, but **humans remain the final decision-makers**.

## Purpose
The OS exists to:
- document business structure and scope
- define processes (SOPs)
- track internal tasks and ideas
- provide consistent context for AI-assisted work

This repository is **documentation-only**.  
It contains no executable code and is not deployed.

## What belongs here
- Company overview and scope
- Operating procedures (SOPs)
- Internal policies and guidelines
- Task inbox and planning notes
- Agent role definitions (conceptual, not code)

## What does NOT belong here
- Client personal data
- Secrets, API keys, or credentials
- Invoices, receipts, or financial records
- Executable code or automation logic

## AI usage principles
- AI systems may read this repository in **read-only mode**
- AI may suggest changes, but never apply them directly
- All changes are reviewed and committed by humans

## Structure
Documentation lives under the `docs/` directory and is organized by topic
(e.g. operations, finance, policies, tasks).
