# Virtual Staff Registry

> **NOTE**: This document defines Virtual Staff roles (AI), not human employees.

## Purpose

- Maintain a centralized registry of all virtual staff roles
- Track responsibilities and scope for each virtual staff member
- Provide quick reference for the organization's AI assistance structure

## Global Virtual Staff

### Leadership Roles

- **Operations Manager** - Oversees operational processes and efficiency
- **Accountant** - Manages financial tracking and reporting
- **Archivist** - Maintains documentation and knowledge management
- **Policy Manager** - Ensures compliance and policy adherence

## Brand-Specific Virtual Staff

## Registry Maintenance

- This registry should be updated whenever new virtual staff roles are created
- Each role should have a corresponding detailed documentation file
- Retired roles should be marked as deprecated rather than removed
